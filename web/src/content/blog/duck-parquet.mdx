# Parquet Data from DuckDB!

This post was generated with API call, and links to data stored as parquet file `instances_silver_agg.parquet`

## Overview of Changes

### 1. Parquet Data Generation

I got data from a csv, put into pandas, than aggregated with duckdb to export as parquet

```python
import pandas as pd
import duckdb
my_date_format = "%Y-%m-%d %H:%M:%S.%f%z"
my_df = pd.read_csv('/mnt/HC_Volume_105622703/projects/duckops/api/src/instance/SILVER_INSTANCES_DATES.csv', delimiter=';', encoding='latin-1', parse_dates=['start_time', 'end_time', 'last_updated'], date_format=my_date_format)
```
#### Testing it works:

```python
duckdb.sql("SELECT * FROM my_df where exchange_name = 'LOG_RIMOWA-CN'")
```
#### Query to parquet:

```python
parquet_path = "/data/local/volume-nbg1-1/projects/duckops/data/raw/instances_silver_agg.parquet"
duckdb.sql("COPY (SELECT DATE_TRUNC('minute', start_time) as ts_min, count(*) as instance_count, max(total_time) as max_total FROM my_df group by ts_min, total_time, order by ts_min) TO '{}' (FORMAT PARQUET)".format(parquet_path))
```

We should find the data stored in path of `parquet_path` variable

#### Reading from parquet:

```python
duckdb.sql("Select * FROM read_parquet('{}')".format(parquet_path))
```
