Welcome to the first interactive DuckOps blog post! 🦆

Astro allows us to seamlessly weave together markdown content with dynamic React, Vue, or native Astro components. To truly achieve our goal of showcasing a Lakehouse architecture, we need our content layer to talk directly to our data engine.

## Fetching from the Backend

Below, we've embedded an Astro component that fetches data directly from our FastAPI backend during the render process. It calls the `/posts/{id}` endpoint and displays the response in a sleek, beautifully styled card.
